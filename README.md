# EcoNest Landing Page

A rapid MVP landing page for the EcoNest project: eco-friendly, sustainable home goods. Includes clean branding, value proposition, and email waitlist signup to validate user interest.

## Features
- Clear value proposition and sustainable goods market positioning
- Beautiful, modern UI
- Founder-branded logo
- Email collection via Formspree for zero-friction waitlist signup
